# About

Teammate is Full stack team messenger web application using with React, Next.js,Auth.js, Covnex, TailwindCSS, React Hooks, React Redux

### Features

```
- Auth with convex
- Create workspace
- Delete workspace
- Edit workspace
- Create channel
- Delete channel
- Edit channel
- Generate invite code
- Regenerate invite code
- Send a message to member
- Leave the message on channel
- Reply message on thread
- Member role change
- Leave the workspace
- Remove memebr from workspace
- Dark theme mode
```

### Install packages

```shell
npm istall
```

### Setup .env file

```js
NEXT_PUBLIC_APP_URL=http://localhost:3000

# Deployment used by `npx convex dev`
CONVEX_DEPLOYMENT=

NEXT_PUBLIC_CONVEX_URL=

```

### Start the app

```shell
npm run dev
```

## Available commands

Running commands with npm `npm run [command]`

| command | description                              |
| :------ | :--------------------------------------- |
| `dev`   | Starts a development instance of the app |
